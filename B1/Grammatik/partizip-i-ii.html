▶ Partizip I
P I Was ist das Partizip I?

Das Partizip I beschreibt eine aktive, gleichzeitige Handlung oder einen dauerhaften Zustand. Es antwortet auf die Frage: „Wie ist es?" oder „Wie macht es das?"

Merksatz: Partizip I = aktiv + gleichzeitig (zur gleichen Zeit!)
P I Bildung
Infinitiv  +  -d

lachen → lachend
schlafen → schlafend
schreien → schreiend
Ausnahme: sein → seiend (selten benutzt)
P I Verwendung
Funktion Beispiel
🔵 Als Adjektiv (vor Nomen) das lachende Kind
🔵 Als Adverb Er läuft lachend herein.
🔵 Relative Handlung Die schlafende Katze liegt auf dem Sofa.
P I Als Adjektiv → Deklination

Wenn Partizip I als Adjektiv vor einem Nomen steht, bekommt es die normale Adjektivendung:

ein schlafendes Kind (Neutrum, Nominativ)
der laufende Mann (Maskulin, Nominativ)
die spielenden Kinder (Plural, Nominativ)
✓ Partizip II
P II Was ist das Partizip II?

Das Partizip II beschreibt eine abgeschlossene Handlung oder einen passiven Zustand. Es wird für Perfekt, Passiv und als Adjektiv benutzt.

Merksatz: Partizip II = fertig / abgeschlossen / passiv
P II Bildung
Regelmäßige Verben:
ge- + Verbstamm + -t
machen → gemacht  |  kaufen → gekauft

Unregelmäßige Verben:
ge- + Verbstamm + -en (oft Vokalwechsel!)
schreiben → geschrieben  |  fahren → gefahren

Verben auf -ieren:
studieren → studiert   (kein ge-!)

Trennbare Verben:
aufmachen → aufgemacht

Untrennbare Verben:
besuchen → besucht   (kein ge-!)
P II Verwendung
Funktion Beispiel
🟣 Perfekt Ich habe den Brief geschrieben.
🟣 Passiv (werden) Das Auto wird repariert.
🟣 Als Adjektiv der gebrochene Arm
🟣 Zustandsbeschreibung Das Fenster ist geöffnet.
⚖️ Vergleich P I vs. P II
▶ Partizip I ✓ Partizip II
Aktiv/Passiv aktiv oft passiv
Zeit gleichzeitig abgeschlossen
Adjektiv das schreibende Kind
(= das Kind schreibt gerade)		 der geschriebene Brief
(= man hat den Brief geschrieben)
Adverb Er geht singend. ❌ kein Adverb
Eselsbrücke:
▶ Ein kochendes Wasser = Wasser, das gerade kocht (aktiv)
✓ Ein gekochtes Ei = Ei, das jemand gekocht hat (fertig, passiv)

Forme den Relativsatz in ein erweitertes Partizipialkonstrukt um. (B1+ Niveau)

Wie geht das?
Relativsatz → Partizip-Konstrukt:
das Kind, das lacht → das lachende Kind (P I = aktiv, gleichzeitig)
der Brief, der geschrieben wurde → der geschriebene Brief (P II = passiv, abgeschlossen)
der Mann, der gerade schläft → der gerade schlafende Mann (mit Adverb!)
